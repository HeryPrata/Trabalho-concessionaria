O programa simula uma concessionaria capaz de comprar, vender e alugar, tanto para clientes quanto para as fabricantes.
O menu disponibiliza comprar, vender, alugar, ver o saldo e finalizar o programa.
o processo de compra e simples e objetivo. Digite modelo e marca, veremos se temos disponivel e logo em seguinda já colocamos todas as informações em tela para que o usuario possa finalizar ou efetuar a compra.
Todo cliente tem um cadastro, e através dele que poderemos saber o saldo do cliente.
